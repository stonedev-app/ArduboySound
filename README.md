# Sample Code for playing sounds on Arduboy

## Requirement

- Arduboy
- PlatformIO

## Usage

1. Install PlatformIO.
2. Clone or download this repository.
3. Connect Arduboy and upload by PlatformIO.
4. Sound plays when A or B button is pressed
